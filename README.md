# Twitter-App
This is a Cloned version of Twitter App using Swift. This is only for personal education purpose.
This clone is featuring most of the features that Twitter has.

# Technologies & Features
- FireBase
- Notification Center
- HTTP, RESTful Api
- JSON Parsing
- MVVM Architecture
- Fully Programmatic UI
- AutoLayout
- CoreGraphics
- UIScrollView
- UITableView
- Protocol Delegate
